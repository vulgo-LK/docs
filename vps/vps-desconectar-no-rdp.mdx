## VPS desconectar no RDP

Quando o Remote Desktop (RDP) apresenta congelamentos intermitentes, isso pode ser causado por diversos fatores, incluindo problemas de rede, configurações de energia do sistema, ou até mesmo configurações específicas do RDP.

## Como resolver o problema do VPS desconectar no RDP

1. Primeiro, instale o arquivo de correção [clicando aqui!](https://drive.google.com/file/d/1_1g7pFfU-bIGl6hCe_FuAsaxBithVsCV/view?usp=sharing) (Não é um arquivo de audio).
2. Após instalar o arquivo basta executar ele normalmente.
3. Pronto, basta fechar e abrir o RDP (Conexão de Área de Trabalho Remota).

<Note>Ele contém uma entrada para desativar o protocolo UDP para conexões de cliente no Remote Desktop Protocol (RDP), o que pode ajudar a resolver problemas de estabilidade em conexões RDP intermitentes. ​</Note>
