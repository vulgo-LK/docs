## Como conectar em VPS windows

1. Primeiro, você deve abrir o Conexão de Área de Trabalho Remota.
2. Para se conectar, você deve pegar os dados do VPS enviados em seu email e WhatsApp.
3. Com isso, basta colocar as informações nos campos designados.

| Campo                               | Descrição                                  |
| ----------------------------------- | ------------------------------------------ |
| Computador:                         | Coloque o IP do VPS                        |
| Nome de Usuário:                    | Administrator                              |
| Permitir salvar minhas credenciais: | Marcar caso queira salvar a senha do VPS   |

<img height="200" src="/images/vps/como-conectar-vps-windows/rdp.png" />

4. Por padrão do nosso sistema o `Nome de Usuário` é **Administrator** em todos os VPS windows.
5. Coloque a senha do seu VPS.

<img height="200" src="/images/vps/como-conectar-vps-windows/senha.png" />

6. Pronto! VPS conectada, tenha um bom aproveito de nossos serviços.
