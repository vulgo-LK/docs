title: Como formatar o seu servidor
description: 'Neste guia, mostraremos como formatar o seu VPS.'
---

## Entrando no painel do cliente

1. Faça login em sua conta no **painel do cliente**.
2. Clique em **Ver Detalhes** para acessar o painel de controle do seu VPS.

<img height="200" src="/images/painelclientes.png" />

## Escolhendo o sistema operacional

1. Role a página para baixo até encontrar a opção **Install**.
2. Adicione o sistema operacional desejado.
3. Defina uma senha para o VPS.
4. Clique no botão **Reinstall** e aguarde o sistema realizar a formação do VPS.

<img height="200" src="/images/painel/formatar-vps/formatandovps.png" />

<Warning>É importante lembrar que, ao formatar o VPS, todos os arquivos serão excluídos e um novo sistema operacional será instalado. Certifique-se de fazer um backup de todos os dados importantes antes de prosseguir com a formatação.</Warning>
